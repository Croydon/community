Package Notes
=============

Many of the libraries packaged by Bincrafters have special features which users might want to know about.  For maintainability reasons, Bincrafters has aggregated all of the package-specific documentation here.  If a Bincrafters package is not listed here, it most likely has no special instructions.  Alternatively, we may simply failed to document something important, so please feel free to open an issue on our `Community Github Repository <https://github.com/bincrafters/community/issues>`_ , or better yet, submit a PR to this page.
